# Property Comparison 
## Parte 1 del laboratorio
#### Link del video de explicación: https://drive.google.com/file/d/156184wVQp_zYErgGRrrFByEv4EbfNMS8/view?usp=drive_link
En esta parte se añadio la opción de añadir propiedades a una tabla de comparacion de maximo 3 propiedades a la vez, dando feedback visual al usuario sobre que propiedades estan siendo
comparadas, destacando las caracteristicas como el mejor precio y area de las propiedades comparadas y la posibilidad de eliminar las propiedades de la tabla y añadir nuevas.
## Definition of Done
| Criterio              | Descripción                                                  |
|----------------------|--------------------------------------------------------------|
| UI de selección      | Checkbox o botón para seleccionar propiedades                |
| Límite Maximo de 3   | No se pueden seleccionar más de 3 propiedades                |
| Tabla de comparación | Vista lado a lado con los detalles de las propiedades        |
| Métricas mostradas   | Precio, dormitorios, baños, área, precio por m²              |
| Mejor resaltado      | Precio más bajo y el área más grande resaltados          |
| Quitar de comparar   | Se pueden deseleccionar propiedades desde la vista de comparación |
| Estado vacío         | Mensaje cuando no hay propiedades seleccionadas              |
